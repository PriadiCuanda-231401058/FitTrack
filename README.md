# FitTrack - Daily Workout Planner

FitTrack adalah aplikasi mobile berbasis Flutter yang dirancang sebagai asisten kebugaran personal untuk membantu pengguna merencanakan, menjalankan, dan memantau latihan harian secara terstruktur. Aplikasi ini mengombinasikan perencanaan latihan, pelacakan progress, serta fitur motivasi agar pengguna dapat berlatih secara konsisten dan terukur.

# Deskripsi Proyek

FitTrack dikembangkan sebagai sistem mandiri (<i>self-contained system</i>) dengan tujuan utama membantu pengguna:

- Menyusun program latihan sesuai tujuan pribadi(gain muscle / lose fat)
- Menjalankan sesi latihan dengan panduan yang jelas
- Memantau progress latihan dan kesehatan (BMI)
- Menjaga konsistensi melalui fitur motivasi berbasis gamifikasi

Aplikasi ini menggunakan Firebase sebagai backend service untuk autentikasi dan penyimpanan data secara real-time, serta mendukung platform Android.

# Tujuan Pengembangan

- Menyediakan aplikasi workout harian yang mudah digunakan
- Membantu pengguna membangun kebiasaan latihan yang konsisten
- Memberikan motivasi berkelanjutan melalui sistem streak, achievement, dan leaderboard
- Menyajikan laporan progress latihan yang informatif dan terukur

# Fitur Utama

## Autentikasi

- Register akun
- Login dengan email & password
- Login dengan google
- Reset password

## Workouts & Exercise

- Daftar latihan berdasarkan tujuan (gain muscle, lose fat), fokus tubuh (dada, lengan, kaki, dll), popularitas
- Exercise list dalam setiap sesi latihan
- Detail exercise dengan video panduan, repetisi atau durasi, timer otomatis

## Pencarian

- Search workout berdasarkan nama

## Motivasi & Gamifikasi

- Streak latihan harian
- Achievement (badge pencapaian)
- Leaderboard (mingguan & bulanan)

## Laporan & Kesehatan

- Progress report latihan
- Ringkasan streak
- BMI check

## Pengaturan

- Edit profil
- Pengaturan akun
- Akses fitur premium
- Logout & delete account

# Target Pengguna

- Pengguna Reguler : Pengguna umum yang ingin melacak dan meningkatkan rutinitas latihan harian.
- Pengguna Premium : Pengguna aktif yang ingin mengakses fitur lanjutan.

# Metodologi Pengembangan

FitTrack dikembangkan menggunakan model Waterfall, dengan tahapan:

1. Requirement Analysis (SRS)
2. UI/UX Design
3. Implementation
4. Testing
5. Deployment & Maintenance
